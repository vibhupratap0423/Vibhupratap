Great! Here's a simple step-by-step guide to create and host your portfolio on GitHub using GitHub Pages:

✅ Step 1: Create a GitHub Account

If you don't have one already, sign up at github.com.

✅ Step 2: Create a New Repository

  1. Go to your GitHub dashboard.

  2. Click the "New" button or go to github.com/new.

  3. Name your repository something like yourusername.github.io
    (replace yourusername with your actual GitHub username — this is important for GitHub Pages to work).

  4. Make it Public.

  5. Check "Add a README file".

  6. Click "Create repository".

✅ Step 3: Add Your Portfolio Files

You can either:

  • Upload HTML/CSS/JS files directly to the repo.

  • Or, clone the repo to your local machine using Git and push your files.

Example HTML file (index.html):

html
<!DOCTYPE html>
<html>
<head>
  <title>My Portfolio</title>
</head>
<body>
  <h1>Hello, I'm Jane Doe</h1>
  <p>Welcome to my portfolio site!</p>
</body>
</html>

✅ Step 4: Enable GitHub Pages

  1. Go to your repository's Settings.

  2. Scroll to "Pages" in the sidebar.

  3. Under "Source", select the branch (main) and root (/) folder.

  4. Click Save.

GitHub will give you a live URL like https://yourusername.github.io.

✅ Step 5: View Your Site

After a few minutes, your portfolio should be live at:
